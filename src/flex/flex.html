Flex grid 1

  • 3 units -> 1 - 2 - 3
  • The orientation of the flex grid is horizontal and all items are on one line
  • The first unit takes up as much space as possible
  • The following units only take up the amount of space the content needs
  • Spacing between the units is gap-2, which means 8 pixels
